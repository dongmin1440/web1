JavaScript

JavaScript 은 HTML 및 CSS 와 함께 월드 와이드 웹 의 핵심 기술 중 하나입니다 . JavaScript는 대화 형 웹 페이지를 가능하게 하며 웹 응용 프로그램 의 필수 부분입니다 . 대부분의 웹 사이트 는 클라이언트 측 페이지 동작에 이 웹 사이트를 사용 하며 모든 주요 웹 브라우저 는 전용 자바 스크립트 엔진 을 사용하여 이를 실행합니다.
